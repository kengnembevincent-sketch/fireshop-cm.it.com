# Fireshop CM

This is the new HTML/CSS/JS structure for the Fireshop CM project.

## Structure Overview
- **HTML**: All HTML files are located in the `html` directory. Each page is created as a separate HTML file.
- **CSS**: Stylesheets are organized in the `css` directory. Ensure to include these in your HTML files for proper styling.
- **JavaScript**: All JavaScript files are found in the `js` directory. They manage the functionality of the website.

## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/kengnembevincent-sketch/fireshop-cm.it.com.git
   ```
2. Navigate to the project directory:
   ```bash
   cd fireshop-cm.it.com
   ```
3. Open the HTML files in your favorite browser to see the project in action.
4. Feel free to modify any files as per your requirements. Ensure to follow the structure laid out above for better maintainability.
